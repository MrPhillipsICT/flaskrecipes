# Flask Recipe Submission Web App

This is a Flask-based web application where users can register, log in, and submit their own cooking recipes.

## Features

- User Registration
- User Login
- Recipe Submission
- View Recipes

## Requirements

- Flask
- MySQL
- Flask-MySQLdb
- Flask-WTF
- Flask-Login
- Flask-Bcrypt
